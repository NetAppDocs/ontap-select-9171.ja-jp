= デプロイ管理者のパスワードを変更する

必要に応じて、コマンド ライン インターフェイスを使用して、Deploy 仮想マシン管理者アカウントのパスワードを変更できます。

.手順
. 管理者アカウントを使用して、デプロイ ユーティリティ CLI にSign in。
. パスワードを変更します:
`password modify`
. 環境に応じて、すべてのプロンプトに応答します。
